# レッスン詳細画面の日付表示削除

## 概要
- レッスン詳細画面（モーダル）において、日付の表示を削除する。
- ユーザーからの要望に基づき、不要な情報を削減して画面をスッキリさせる。

## 仕様
### 機能要件
- レッスン詳細モーダル内の「日付」項目（アイコン、ラベル、日付、曜日）を非表示にする。

### UI/メッセージ
- レッスン詳細モーダルから日付に関連するセクションが消える。
- その他の項目（時間、場所など）のレイアウトは維持する。

### 制約・前提
- 既存の `selectedItem.date` や `selectedItem.dayOfWeek` のデータ自体は削除しない。
- 一覧画面の曜日表示などは現時点では変更しない。

## 実装計画
### 使用するクラス・ファイル
- View: `src/view.html`

### 処理フロー
1. `src/view.html` 内の `selectedItem` が表示されるモーダル部分を探す。
2. 日付を表示している React コンポーネントの JSX ブロック（`Icon name="calendar"` が含まれる部分）を削除またはコメントアウトする。

### 技術的な判断・注意点
- 単に JSX を削除するだけで要件を満たせる。
- スタイリング（`space-y-4` など）が維持されることを確認する。
